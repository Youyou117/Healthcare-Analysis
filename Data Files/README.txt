This folder contains 3 zip files from CCHS cycles 1.1, 2.1 and 3.1 (Original raw data). These zip files contain the unedited original Statistics Canada public-use datafiles. Associated documentation and licence agreements are provided in the respective 'documentation' folders within the zip files. 

Note that the use of these files must be done with respect to the terms and conditions of the Statistics Canada Open Licence (link: https://www.statcan.gc.ca/eng/reference/licence). Please consult the licence agreements provided here.

The R datasets (Converted RData files) are also provided here (this was not part of the original release), which are converted from the entire original HS.txt datasets in those zip files. All the variables and entries should be available in these RData files.
